# Rencana Perbaikan Overlap Navbar di Login & Profile

## Masalah
Halaman Login dan Profile memiliki konten (card) yang bisa tumpang tindih dengan navbar fixed di viewport pendek, karena:
1. `overflow-hidden` di container utama memblokir scroll
2. `mt-16` (64px) di card kurang dari tinggi navbar (~76px)

## File yang Diperbaiki

### 1. `app/login/page.tsx`
- **Baris 111**: Ganti `overflow-hidden` → `overflow-y-auto` agar konten bisa di-scroll di viewport pendek
- **Baris 198**: Hapus `mt-16` dari card
- **Baris 111**: Tambahkan `pt-24` ke outer container untuk memberi ruang di bawah navbar

### 2. `app/user/profile/page.tsx`
- **Baris 168**: Ganti `overflow-hidden` → `overflow-y-auto`
- **Baris 245**: Hapus `mt-16` dari card
- **Baris 168**: Tambahkan `pt-24` ke outer container

## Alasan Perubahan
- `pt-24` (96px) > tinggi navbar (~76px), sehingga konten tidak akan tumpang tindih
- `overflow-y-auto` memungkinkan scroll jika konten lebih tinggi dari viewport
- Card tetap di-center secara vertikal oleh `flex items-center justify-center` di dalam area yang sudah dikurangi padding top

## Verifikasi
- Jalankan `npx next build` untuk memastikan tidak ada error
- Cek visual di browser dengan resize viewport ke ukuran kecil (< 700px) untuk memastikan tidak ada overlap
