# Rencana Perbaikan Admin Panel NeonCharge

## Ringkasan
Perbaikan menyeluruh admin panel: CRUD stasiun, KPI dashboard, keuangan detail, riwayat transaksi, dan download PDF.

---

## 1. Install jsPDF untuk download PDF
- Install `jspdf` dan `jspdf-autotable` untuk generate PDF profesional
- `npm install jspdf jspdf-autotable`

---

## 2. Fix CRUD Stasiun (`app/admin/stasiun/page.tsx`)

**Masalah:** Tombol "+ Tambah Stasiun", Edit, dan Delete tidak memiliki onClick handler.

**Perbaikan:**
- Ubah data stasiun dari fetch statik ke state management (useState + localStorage)
- Tambah Dialog untuk **Tambah Stasiun** (ID, Lokasi, Tipe Konektor, Status)
- Tambah Dialog untuk **Edit Stasiun** (pre-fill data)
- Tambah Dialog konfirmasi **Delete Stasiun**
- Gunakan `useRef` & `localStorage` untuk persist data (sama seperti pattern di override page)

---

## 3. Fix KPI Dashboard (`app/admin/dashboard/page.tsx`)

**Masalah:** Text di KPI cards perlu perbaikan styling.

**Perbaikan:**
- Label (text-xs text-slate-400) → ganti ke `font-semibold text-slate-500` supaya lebih terbaca
- Value (text-2xl font-bold) → pertahankan bold, tambah `text-left` eksplisit
- Pastikan semua card menggunakan `text-left` agar tidak center

---

## 4. Enhance Keuangan Page (`app/admin/keuangan/page.tsx`)

**Masalah:** Hanya placeholder "Area Grafik Pendapatan".

**Perbaikan — tambahkan:**
- Ringkasan pendapatan (Total, Bulan Ini, Rata-rata/Hari)
- Tabel pendapatan per stasiun (Lokasi, Total Transaksi, Total kWh, Pendapatan)
- Breakdown per tipe konektor
- Tombol "Download Laporan PDF"

---

## 5. Tambah Riwayat Transaksi (`app/admin/transaksi/page.tsx`)

**Halaman baru** dengan:
- Tabel riwayat transaksi (ID, Waktu, Stasiun, User, kWh, Harga, Status)
- Filter by status (Semua, Selesai, Proses, Dibatalkan)
- Data mock realistis (10-15 transaksi)
- Tombol "Download PDF" untuk export riwayat

**Update layout admin** (`app/admin/layout.tsx`):
- Tambah menu "Transaksi" ke sidebar

---

## 6. PDF Download (`lib/pdf-utils.ts`)

**File baru** berisi fungsi generate PDF:
- `generateLaporanKeuangan()` — laporan keuangan profesional dengan header, logo teks, tabel, total
- `generateRiwayatTransaksi()` — export tabel transaksi ke PDF
- Format: A4, header "NeonCharge - Laporan Operasional", tanggal, watermark
